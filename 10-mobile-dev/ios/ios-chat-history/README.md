# ios-chat-history

A hands-on skill for learning ios-chat-history in Swift/SwiftUI.

## Prerequisites

- Xcode 15+ (Swift 5.9+)
- macOS 14+
- Basic Swift knowledge

## Quick Start

```bash
# Install dependencies
make setup

# Run examples
make examples

# Run tests
make test
```

## Learning Path

1. **Read** `SKILL.md` for quick overview
2. **Run** examples with `make examples`
3. **Study** code in `Sources/Examples/`
4. **Deepen** understanding with `docs/`
5. **Practice** with `exercises/`
6. **Verify** with `make test`
